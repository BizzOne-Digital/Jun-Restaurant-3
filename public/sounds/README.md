# Notification Sounds

Place your notification audio file here:

  public/sounds/new-order.mp3

Requirements:
- Format: MP3 (best browser compatibility)
- Duration: short beep/chime, 1–3 seconds recommended
- The file MUST be named exactly: new-order.mp3

The admin portal polls for new paid orders every 30 seconds.
When a new paid order is detected, this sound plays automatically.

Free sources for notification sounds:
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org
- https://notificationsounds.com
